Welcome to the Pac-Man Game

Pac-Man is a classic arcade game where you control a yellow character that eats dots while avoiding ghosts.

Rules:

  • Control Pac-Man using arrow keys to navigate the maze.
  • Eat all the dots in the maze to advance to the next level.
  • Avoid the ghosts. If a ghost catches Pac-Man, you lose a life.
  • You have a limited number of lives, so be careful!

Instructions:

Use the arrow keys (up, down, left, right) to move Pac-Man through the maze. Eat all the dots to clear the level. The game gets progressively harder with more ghosts as you advance.

Game Winning:

To win the game, you must clear all the levels by eating all the dots without losing all your lives to the ghosts.
